# PETS Forward Track Record

This repository is a public, time-stamped archive of weekly PETS outputs.

**Update cadence:** Weekly (Saturday)  
**Method:** Fully systematic. No discretionary selection of symbols.  
**Note:** This is a verification archive; execution may occur on a separate broker.

## Folders

- `signals/` – `signals_YYYY-MM-DD.csv`
- `nav/` – `nav_YYYY-MM-DD.csv`
- `performance/` – `performance_YYYY-MM-DD.json`
- `weekly_summary/` – `summary_YYYY-MM-DD.md`

## Disclosure Philosophy

This repo is a **verification layer**, not research transparency.

Only minimal columns are published publicly to preserve IP while allowing timestamp integrity.
